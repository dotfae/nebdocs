<Accordion title="General" icon="gear">
  - **Chest Closer**: Automatically closes chests in dungeons.
  - **Bigger Secret Hitboxes**: Expands buttons, levers, chests, and essence hitboxes to a full-sized block.
  - **M3 Fire Freeze Timer**: Shows a timer for Fire Freeze.
  - **Auto Use Fire Freeze**: Automatically uses Fire Freeze.
  - **Terminator Clicker**: Spams left click when holding right click.
  - **Gyro Radius**: Displays the radius for the Gyrokinetic Wand.
  - **Gyro Radius Only in Thirdperson**: Shows the radius for the Gyrokinetic Wand only in third person.
  - **Blessings Display HUD**: Shows current blessings.
  - **Auto Send Message on Teammate Death**: Sends a message when a teammate dies.
  - **Message to send**: Custom message for "Auto Send Message on Teammate Death".
  - **Mimic Message**: Sends a message on Mimic death.
  - **Mimic Message Text**: Custom message for Mimic Message.
  - **No DT**: Automatically starts a new dungeon run.
  - **Auto GFS**: Automatically gets ender pearls from sacks.
  - **Block Interaction**: Allows you to throw pearls even if interacting with a block.
  - **Vertical Jerry**: Changes Jerry Gun velocity direction. (Currently disabled due to new AC update)
  - **Bonzo Movement Helper**: Improves movement with Bonzo Staff. (Currently disabled due to new AC update)
  - **Tank/Healer Ultimate Notification**: Sends a message when you need to use the ultimate ability.
  - **Tank/Healer Auto Use Ultimate**: Automatically uses the ultimate ability on Tank/Healer at certain moments.
  - **No Death Animation**: Removes death animation.
  - **Auto Stair Align Keybind**: Keybind to automatically align you. (Currently disabled due to new AC update)
  - **Auto Stair Align**: Automatically aligns you without pressing a keybind. (Currently disabled due to new AC update)
  - **Doorless**: Allows you to skip through a wither / blood door using ender pearls. (Currently disabled due to new AC update)
  - **Doorless Motion**: Pushes you forward after using Doorless. (Currently disabled due to new AC update)
  - **Auto Left Click Superboom Swap**: Automatically presses left click after swapping on Superboom/Infinityboom TNT.
  - **Auto Mask Swap**: Swaps masks (Bonzo, Spirit). (Currently disabled due to new AC update)
</Accordion>

<Accordion title="Usage Notes" icon="notes-medical">
  - **/n dungeons brush**: toggles brush edit mode (allows you to place/remove blocks)
  - **/n dungeons brush select**: sets block for brush (block ID/block name)
  - **/n dungeons brush load** : reloads current brush config
</Accordion>
