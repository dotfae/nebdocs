<Accordion title="Visuals" icon="eye" iconType="duotone">
  - **Anti Blindness**: Removes blindness effects from your screen.
  - **Hide Portal Overlay**: Hides the purple portal overlay effect from your screen.
  - **Hide Fire Overlay**: Removes the fire overlay effect when taking fire damage.
  - **Hide Water Overlay**: Removes the water overlay effect when underwater.
  - **Full Bright**: Provides maximum brightness regardless of in-game lighting conditions.
</Accordion>

<Accordion title="Custom Third Person View" icon="video" iconType="duotone">
  - **Custom Third Person View**: Enables customizable third person camera perspective.
  - **Distance**: Adjusts the camera distance from your character in third person view.
  - **Clip through blocks**: Allows the camera to pass through blocks instead of being blocked by them.
  - **No front view**: Disables switching to front view when toggling perspective, keeping you in third person back view.
</Accordion>

<Accordion title="Custom Inventory Character" icon="user" iconType="duotone">
  **Custom Inventory Character**: Enables custom character display in inventory GUI.

  - **Path (URL or Filename with extension)**: File path or URL for custom character image. You can add custom assets here: .minecraft/Nebula/Visuals/characters
  - **X**: Horizontal position adjustment for the custom character.
  - **Y**: Vertical position adjustment for the custom character.
  - **Scale**: Size scaling for the custom character display.
</Accordion>

<Accordion title="Custom GIF HUD" icon="image" iconType="duotone">
  - **Custom GIF HUD**: Displays a GIF on your screen.
  - **Custom GIF URL**: The URL of the GIF to display.
</Accordion>

<Accordion title="General" icon="gear" iconType="duotone">
  - **Hide Players**: Makes other players invisible.
  - **Click Through Hidden Players**: Allows clicking through hidden players as if they weren't there.
  - **Hide Players Mode**: ⤵

  <Expandable title="Hide Players Options">
    - **All**: Hides all players.
    - **Near You**: Hides players within a certain radius.
  </Expandable>
  - **Radius To Hide Players In**: Distance in blocks within which players will be hidden.
  - **Highlight Glowing Mushrooms**: Makes glowing mushrooms more visible with ESP.
</Accordion>

<Accordion title="Scale" icon="expand" iconType="duotone">
  - **NPC Scale**: Adjusts the size of NPCs in the game.
  - **Player Scale**: Adjusts the size of other players.
  - **Only Yourself**: When enabled, scaling changes only apply to your own character.
  - **Mob Scale**: Adjusts the size of mobs and monsters.
</Accordion>

<Accordion title="Thyst Hider" icon="eye-slash" iconType="duotone">
  - **Thyst hider**: Hides thyst.
</Accordion>

<Accordion title="Freecam" icon="camcorder" iconType="duotone">
  - **Toggle Freecam**: Keybind to toggle freecam on/off depending on previous state.
  - **Freecam speed**: Adjusts the speed of your movement in freecam.
  - **Freecam Speed Vertically**: Adjusts the vertical (up/down) speed of your movement in freecam.
</Accordion>

<Accordion title="Perspective Toggle" icon="camera" iconType="duotone">
  - **Perspective Toggle**: Activates third person perspective on key press.
  - **Perspective Toggle Mode**: ⤵

  <Expandable title="Toggle Modes">
    - **Hold**: Third person view is active only while holding the keybind.
    - **Toggle**: Third person view toggles on/off with each keybind press.
  </Expandable>
  - **Perspective Keybind**: Keybind to toggle third person perspective.
</Accordion>

<Accordion title="Custom ESP" icon="radar" iconType="duotone">
  - **Custom ESP**: Enables custom ESP to highlight specific entities.
  - **Custom ESP Mode**: ⤵

  <Expandable title="ESP Modes">
    - **Outline**: Creates a colored outline around targeted entities.
    - **Box**: Draws a box outline around targeted entities.
    - **Chams**: Renders a solid color overlay on targeted entities.
    - **Glow**: Adds a glowing outline effect to targeted entities.
  </Expandable>
  - **Custom ESP Color**: Sets the color for ESP.
  - **Render Players**: Shows ESP on other players.
  - **NPC Name List**: List of specific NPC names to target with ESP. Separate names with a semicolon.
</Accordion>

<Accordion title="Custom Block Highlight" icon="cube" iconType="duotone">
  - **Custom Block Highlight**: Enables custom highlighting for specific blocks.
  - **Highlight Style**: ⤵

  <Expandable title="Highlight Styles">
    - **Static**: Uses a single color for highlighting.
    - **Rainbow**: Cycles through rainbow colors for the highlight.
    - **LSD**: Cycles through colors rapidly for a flashing effect.
  </Expandable>
  - **Depth**: Enables depth testing for block highlights.
  - **Speed**: Controls the speed of animated highlight effects.
  - **Thickness**: Adjusts the thickness of highlight outlines.
  - **Animated BB size**: Enables animated bounding box sizing.
  - **Edges Color**: Sets the color for highlight edges.
  - **Faces Color**: Sets the color for highlight faces.
  - **Etherwarp Display**: Shows Etherwarp teleportation locations.
  - **Etherwarp Success Color**: Color for successful Etherwarp locations.
  - **Etherwarp Fail Color**: Color for invalid Etherwarp location.
</Accordion>

<Accordion title="Chest ESP" icon="treasure-chest" iconType="duotone">
  - **Chest ESP**: Enables ESP on chests to make them easier to locate.
  - **Chest ESP Mode**: ⤵

  <Expandable title="ESP Modes">
    - **Box**: Draws a box outline around chests.
    - **Glow**: Adds a glowing outline effect to chests.
  </Expandable>
  - **Only in Crystal Hollows**: Restricts chest ESP to only work in Crystal Hollows.
</Accordion>

<Accordion title="Ghost HP" icon="ghost" iconType="duotone">
  - **Ghost HP**: Displays name tag and health bars for ghost mobs.
</Accordion>

<Accordion title="Item Animation" icon="hand" iconType="duotone">
  - **Item Animations**: Enables custom animations for items held in hand.
  - **No Equip Reset**: Prevents the item equip animation from resetting.
  - **No Equip Reset on Slot Change**: Prevents the item equip animation from resetting when changing hotbar slots.
  - **Size**: Adjusts the size of the held item.
  - **Scale Swing**: Adjusts the speed of your swing.
  - **X**: Horizontal position adjustment for the held item.
  - **Y**: Vertical position adjustment for the held item.
  - **Z**: Depth position adjustment for the held item.
  - **Yaw**: Rotation around the vertical axis for the held item.
  - **Pitch**: Rotation around the horizontal axis for the held item.
  - **Roll**: Rotation around the forward/backward axis for the held item.
  - **Speed**: Controls the speed of swing animations.
  - **Ignore Haste**: Disables haste effects on swing animation speed.
  - **No Block**: Removes blocking animations when using items.
</Accordion>

<Accordion title="Custom Scoreboard" icon="list" iconType="duotone">
  - **Custom Scoreboard**: Overrides Skyblock's scoreboard to Nebula's version.
  - **Scoreboard Corner Roundness Radius**: Adjusts the roundness of scoreboard corners.
  - **Scoreboard BG Blur**: Enables background blur effect for the scoreboard.
  - **Scoreboard BG Blur Strength**: Controls the strenght of the background blur.
  - **Scoreboard BG Color**: Sets the background color of the scoreboard.
  - **Scoreboard Outline Color**: Sets the outline color of the scoreboard.
  - **Scoreboard Outline Width**: Adjusts the width of the scoreboard outline.
  - **Scoreboard Shadow Color**: Sets the color of the scoreboard's drop shadow.
  - **Scoreboard Title Text**: Custom text to display as the scoreboard title.
</Accordion>
