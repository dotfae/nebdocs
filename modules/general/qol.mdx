<Accordion title="Auto Harp" icon="music">
  - **Auto Harp**: Automatically completes harp.
  - **Delay**: Delay between clicks.
  - **Delay on double**: Delay between clicks.
  - **Click mode (Left/Middle)**: Left or middle click for Auto Harp.
  - **Close on failure**: Closes if not 100% accuracy.
</Accordion>

<Accordion title="Auto Experiment" icon="table-picnic">
  - **Superpairs Helper**: Solver for Superpairs.
  - **Auto Experiment**: Automatically completes Experiments.
  - **Delay**: Delay between clicks.
  - **Auto close**: Automatically closes the GUI after completing experiment.
  - **Metaphysical Serums consumed**: Amount of consumed Metaphysical Serums.
</Accordion>

<Accordion title="Auto Join" icon="server">
  - **Auto Join Hypixel On Client Start**: Joins Hypixel on client start.
  - **Auto Join Skyblock**: Joins Skyblock when connected to Hypixel.
</Accordion>

<Accordion title="Sound Silencer" icon="music-slash">
  - **Sound Silencer**: Toggles the sound silencer.
  - **Silence Hyperion Sounds**: Removes Hyperion sounds.
  - **Silence AOTV/E Sounds**: Removes AOTV/E sounds.
  - **Silence Giant's Sword**: Removes Giant's Sword sounds.
  - **Silence Endermen Sounds**: Removes Endermen sounds.
  - **Silence Ender Dragon Sounds**: Removes Ender Dragon sounds.
  - **Silence Ghast Sounds**: Removes Ghast sounds.
</Accordion>

<Accordion title="Hotbar Swapper" icon="sitemap">
  - **Hotbar Swapper Keybind**: Keybind to start and stop the hotbar swapper.
  - **Dungeons Class Hotbar Swapper Keybind**: Keybind to start and stop the hotbar swapper for Dungeon Classes.
  - **Next Hotbar Preset Keybind**: Selects the next hotbar preset.
  - **Hotbar Preset**: Selects the current hotbar preset.

    <Expandable title="Commands">
      - **/n qol hotbar save [index]**
      - **/n qol hotbar select [index]**
      - **/n qol hotbar dungeons save [class]**
    </Expandable>
  - **Swap Delay**: Delay between swapping items.
</Accordion>

<Accordion title="Misc" icon="sparkles">
  - **Nick Protect**: Toggles Spof.
  - **Spoofed nickname**: Write a custom nick.
  - **Auto Conversation**: Automatically clicks "Yes" in dialogue with NPCs.
  - **Auto AH Reopen**: Reopens the auction house if you get "escrow not found" error.
  - **Pickup Stash Now**: Picks up stash.
  - **Auto Sprint**: Toggles sprint.
  - **Ban Detector**: Detects banned players in your lobby.
  - **Auto Party Chat**: Automatically types /chat p on joining or creating a party.
  - **Auto Cake Clicker**: Smoothly rotates and clicks on cakes nearby if the cake is not on cooldown.
  - **Hold Pressed Buttons**: Toggles holding of pressed buttons.
  - **Auto Use Rogue Sword**: Automatically uses Rogue Sword.
  - **Auto Use Rogue Sword Only in Dungeons**: Automatically uses Rogue Sword only in dungeons.
  - **Auto Minion Crafter**: Automatically crafts maxed minions (Tier 11) from resources from the bazaar.
  - **Peak Aura**: Automatically clicks on candy and gifts during Halloween in a special minigame.
  - **Primal Fear Notifier**: Notifies you when you spawn a Primal Fear while any macro is enabled.
  - **Jerry Box Opener**: Automatically opens Jerry Boxes.
  - **Delay**: Delay between opening Jerry Boxes.
</Accordion>

<Accordion title="Cursor Reset" icon="i-cursor">
  - **No Cursor Reset**: Prevents your cursor from resetting between GUIs.
</Accordion>
