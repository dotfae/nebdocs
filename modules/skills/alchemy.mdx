<Accordion title="General" icon="flask-round-potion">
  - **Alchemy Macro Keybind**: Toggles Alchemy Macro
  - **Upgrade ingredient**: (You can check prices using skyhelper bot in [commands](https://discord.com/channels/1188447621040328854/1332418751131291832) channel)

    <Expandable title="ingredients">
      - **None**
      - **Glowstone Dust**
      - **Enchanted Glowstone Dust**
      - **Enchanted Glowstone**
    </Expandable>
  - **Mode**:

    <Expandable title="Modes">
      - **Automatic**: Fully automatic macro
      - **Manual**: Semi-auto mode
    </Expandable>
  - **Auto Bottle Filler**: Keybind to start filling glass bottles, you should stand in water above hoppers and have some empty glass bottles in hotbar
</Accordion>

<Accordion title="Options" icon="sparkles">
  - **Auto put water bottles**: Automatically puts new water bottles in brewing stands
</Accordion>

<Accordion title="Manual Mode" icon="sparkles">
  - **Auto put ingredients**: Automatically puts ingredients in brewing stands
  - **Auto pick up finished potions**: Automatically picks up finished potions from brewing stands
</Accordion>

<Accordion title="Build Helper" icon="building">
  - **Build Overlay**: Schematica for alchemy macro
  - **Set Build Position**: Sets position for build overlay
</Accordion>
